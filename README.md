# MODELOS-IA

## Objetivo
Verificar la calidad de los datos y definir modelos de Machine Learning e Inteligencia Artificial para el análisis estructural.

## Actividades
- Limpieza de datos
- Manejo de valores nulos
- Análisis de correlación
- Visualización de datos
- Selección de algoritmos de ML/IA
- Evaluación mediante métricas

## Herramientas
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Métricas de Evaluación

Para modelos de regresión se utilizarán:

- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² (Coeficiente de Determinación)

## Conclusiones

El análisis exploratorio permitió verificar la calidad de los datos y analizar la relación entre las variables que influyen en la resistencia del concreto. Se propone el uso de Random Forest y Redes Neuronales para desarrollar modelos predictivos capaces de estimar la resistencia a compresión con alta precisión.
